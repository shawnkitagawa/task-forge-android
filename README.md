# Task Forge 

Task Forge is a modern Android Todo application built with **Kotlin and Jetpack Compose**.

The goal of this project is to practice modern Android architecture including:

- MVVM architecture
- StateFlow state management
- Jetpack Compose UI
- Clean architecture principles

---

## Features

- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks (All / Completed)

---

## Tech Stack

- Kotlin
- Jetpack Compose
- ViewModel
- StateFlow
- Coroutines

---

## Architecture

The project follows **MVVM architecture**.

UI (Compose)
↓
ViewModel (StateFlow)
↓
Repository
↓
Data source
